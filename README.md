# Desbravadores Finance

Sistema open source para controle financeiro de clubes de Desbravadores.

O projeto foi criado com o objetivo de oferecer uma solução simples, acessível e gratuita para organização do orçamento do clube, promovendo transparência, responsabilidade e melhor tomada de decisão financeira.

---

## 🎯 Objetivo

Permitir que a diretoria do clube:

- Registre entradas e saídas financeiras  
- Acompanhe o saldo atualizado  
- Organize categorias de receitas e despesas  
- Tenha visão clara da situação financeira  
- Planeje metas e arrecadações com mais segurança  

---

## 💡 Proposta

A proposta é disponibilizar uma ferramenta leve, prática e adaptável à realidade de clubes pequenos e médios, sem custos de infraestrutura complexa.

O sistema utiliza tecnologias simples e acessíveis, permitindo que qualquer clube possa implementar e utilizar sem depender de soluções pagas.

---

## 🚀 Tecnologias

- React Native (com Expo) para os apps móveis (Android e iOS) e Web
- TypeScript
- Supabase (PostgreSQL + Autenticação)

---

## 🔭 Visão Futura

Nesta nova versão (v2), estamos implementando as seguintes necessidades escaláveis:
- Controle de múltiplos clubes com segurança de dados isolados  
- Permissões por cargo (diretor, tesoureiro, etc.)  
- Relatórios automáticos  
- Sugestões inteligentes de arrecadação  
- Versão instalável como aplicativo (PWA)
